Functional Interface Rules

Quick reference · Oracle javadoc

The SAM rule

A functional interface has exactly one abstract method (SAM = Single Abstract Method).

What does NOT count as abstract

  • default methods — they have a body
  • static methods — they have a body
  • Methods overriding Object public methods (e.g. toString)

Instantiation

A functional interface can be implemented via:

  • Lambda expression
  • Method reference (System.out::println)
  • Constructor reference (ArrayList::new)
  • Anonymous inner class (verbose; avoid when lambda works)

Annotation

@FunctionalInterface
public interface MyOp {
    int apply(int a, int b);
}

@FunctionalInterface is optional but recommended — the compiler errors if you add a second abstract method.
