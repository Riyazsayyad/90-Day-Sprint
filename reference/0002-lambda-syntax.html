Lambda Expression Syntax

Quick reference · Oracle tutorial

Shape

(parameters) -> expression
(parameters) -> { statements; }

Parameter forms

CaseSyntax
Zero params() -> expr
One param (inferred)x -> expr or (x) -> expr
One param (typed)(String s) -> expr
Multiple params(a, b) -> expr — parens required

If you declare a type for one parameter, declare types for all parameters.

Body forms

FormReturns?Example
ExpressionYes — value is returnedx -> x * 2
BlockYes — needs returnx -> { return x * 2; }
Void statementNo — single void call OKs -> System.out.println(s)
Void blockNo — no return value() -> { System.out.println("hi"); }

Target typing

The compiler infers parameter types from the expected functional interface — the variable type, method parameter, or cast on the left.

Predicate<String> empty = s -> s.isEmpty();
//                          ^ inferred as String

Anonymous class → lambda

// Before
Runnable r = new Runnable() {
    public void run() { System.out.println("go"); }
};

// After
Runnable r = () -> System.out.println("go");
