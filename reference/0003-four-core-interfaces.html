Predicate · Function · Consumer · Supplier

java.util.function · Oracle package

InterfaceInputOutputMethodQuestion it answers
Predicate<T>Tbooleantest(T)Does it pass?

Consumer<T>Tvoidaccept(T)Do something with it
Supplier<T>noneTget()Give me a value

Stream mapping

list.stream()
    .filter(x -> x % 2 == 0)   // Predicate — keep or drop
    .map(x -> x * x)           // Function — transform
    .forEach(System.out::println); // Consumer — act on each

Lambda examples

Predicate<String> longEnough = s -> s.length() > 5;
Function<String, Integer> len = String::length;
Consumer<String> shout = s -> System.out.println(s.toUpperCase());
Supplier<Double> random = () -> Math.random();

Default methods (worth knowing)

  • Predicate.and / or / negate — combine tests
  • Function.andThen / compose — chain transforms
  • Consumer.andThen — run two side effects in sequence

When to use primitives

Prefer IntPredicate, ToIntFunction, etc. in hot paths — avoids boxing overhead. See specialized variants.
